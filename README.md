# DocumentManagement

Pharmaceutical DMS demo with 21 CFR Part 11 oriented features: audit trail, e-signatures, role-based access, versioning, and configurable workflows. Not production legal advice.

## Run

```
npm install
npm run dev
```

## Build

```
npm run build
```

## Deploy

```
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-83312b74
```
